# Code of Conduct

This project aims to be a respectful, practical place to discuss and improve the extension.

## Expected Behavior

- Be clear and constructive.
- Discuss technical tradeoffs directly without personal attacks.
- Respect privacy and do not post credentials, private endpoints, or user data.
- Assume maintainers and contributors are working in good faith.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information without consent.
- Repeatedly derailing issues or pull requests away from the project topic.
- Posting secrets or exploit details in public issues.

Maintainers may edit, hide, or remove comments and contributions that violate these rules.
